# cepik-api-client

A modern TypeScript/JavaScript library for retrieving data from the public API of the Polish National Vehicles Database (CEPIK).

## 📝 Types and Enums

### Enums

```typescript
import {
} from "nbp-api-client";
```

### Types

```typescript
import type {
} from "nbp-api-client";
```

## 🧪 Testing

```bash
npm run test
```

## 🔍 Linting

```bash
npm run lint
```

## 🏗️ Building

```bash
npm run build
```

The compiled code is located in the `dist/` directory.
